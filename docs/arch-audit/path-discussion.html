Path 5 — User clicks through to a discussion

from the Hub → GET /hub/<forum-slug>/<topic-slug>/

Same app as Path 3 (bb-mirror), now with two URL segments → the single-topic view. The thread, its replies and attachments all come from Postgres looth.forums in a handful of joined queries. Viewer identity is again the cached WP whoami.
