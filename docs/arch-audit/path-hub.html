Path 3 — User opens the Hub

GET /hub/  (legacy /forum/ /forums/ /forums-poc/ all 301 → /hub/)

“The Hub” is the bb-mirror app — a standalone forum read surface backed by Postgres looth.forums. It’s a mirror: bbPress in WordPress remains the write origin, and a sync feeds Postgres (see Path 5). Identity comes from a cached loopback to WP.
