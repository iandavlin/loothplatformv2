Path 4 — User clicks through to a CPT

GET /post-imgcap/<slug>/ (also /article/ /video/ /sponsor/ /loothprint/ /document/<id>/ …)

Managed content types are strangled: nginx routes the permalink to the archive-poc standalone renderer, which serves a pre-materialized layout. Only on a cache miss does it hand back to WordPress, where lg-layout-v2 builds the page from the _lg_layout_v2 post-meta. Two render engines, one URL.
