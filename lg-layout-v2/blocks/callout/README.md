# callout

One block, four role-named variants. Hard-left-border container for
asides, lists, files, and link stacks. Body is rich-text (TinyMCE) so
authors compose paragraphs / lists / links normally; the variant supplies
the visual treatment.

| Variant   | Use                          | Look                                            |
|-----------|------------------------------|------------------------------------------------|
| `editor`  | asides, takeaways, emphasis  | amber border, soft gradient, serif body         |
| `people`  | guests, contributors         | sage border, list rows get dashed separators    |
| `files`   | downloads, PDFs, templates   | amber boxed panel, list rows get download glyph |
| `data`    | parts, affiliate links       | ink rail, sans body, dense rows                 |

Style maps to the role on purpose — the reader learns the grammar after
a couple of articles and the color does navigation work without labels.

Edit `title` and `html` inline. Variant flip via the FE editor pill.
